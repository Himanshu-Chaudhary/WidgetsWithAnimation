The main function is in WidetWithAnimation Class.
This program takes no arguments from the user.
There are two buttons and two combobox in GUI.

How to use the Program
-Start/Stop button allows to pause and play animation.
-Resume Buttom Allows to reset the animation with user-defined values of Number of Spheres and Circles. 
-The start button needs to be pressed to play the animation again
